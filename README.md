# QA Portfolio Project – SauceDemo Manual Testing

## Project Overview

This repository demonstrates a complete manual testing process for the public **SauceDemo** web application.

The objective of this project is to showcase practical QA skills using industry-standard testing artifacts that reflect a real-world testing engagement.

## Application Under Test

* **Application:** SauceDemo
* **Type:** Public Demo E-commerce Web Application
* **Testing Type:** Manual Functional Testing

## Scope

This repository includes:

* Requirement Analysis
* Test Plan
* Test Scenarios
* Test Cases
* Requirement Traceability Matrix (RTM)
* Bug Reports
* Test Summary Report
